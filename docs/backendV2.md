# 后端V2 优化细节和规则
## 1 规则
- 30m,4h 不开单之间忽略K线检查。
- 三个账户：短期账户，中期账户，长期账户
- 价格突破1h上下轨后开长期单，中期单，和短期单  去5m周期震荡结构寻找X2X3-Xn的交易机会 
- 价格突破15m上下轨后开中期单，短期单 去5m周期震荡结构寻找X2X3-Xn的交易机会
- 价格突破5m上下轨后开短期单 去1m周期震荡结构寻找X2X3-Xn的交易机会
- 所有周期开单，都必须在1m ，5m 周期震荡结去寻找开单点位和机会。如：价格运行在1h上轨外，那就需要等5m 最近震荡结构 出现Y2关键点确认，去预判Y2Y3的点位做空。价格如果不在1h上轨外，而在15m上轨外，那就需要等5m 出现Y2关键点确认，去预判Y2Y3的点位做空。也就是1h，15m周期出现价格运行在上轨外或者下轨外，开单限制开启，然后在等5m出现Y1关键点确认，去预判Y2Y3的点位做空。(预判点位就是，确认Y1后，价格再次运行到上轨外面，这个位置附近就会诞生潜在的Y2，Y3也是同理，等Y2确认，价格再次运行到上轨外面，这位置附近往往就会诞生潜在的Y3，Yn也是同理，但总有被打破的时候，就算价格长时间运行在轨外，并强势上涨，这样就会打破震荡结构形态，最后出现的关键点就是新结构体的Y1，这种时候往往就是开单后打止损了，也有可能一根k打了止损价格又下来了。预判点位去开单就算赌行情的转折，转折了就盈利，失败就止损)。X和Y关键点 都是同理
- 开单限制规则：价格触及上轨后对应周期的开单限制开启，当价格触及中轨后，对应周期的开单限制关闭。所有周期开单限制默认关闭，触及上下轨后开启
- 5m 开单限制开启后，需要判断价格离15m和1h的上下轨距离，如果距离很接近了，那就需要等价格触及15m和1h的上下轨，再开单，且需要在X2X3-Xn的点位开单。Y2Y3-Yn的点位开单
